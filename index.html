Project

Psychiatric Patient Tracking Web Application

The Psychiatric Patient Tracking Web Application helps healthcare providers track and manage psychiatric patients efficiently. It includes features such as:

  • LINE Notify: Sends alerts when a patient misses an appointment, allowing for timely follow-up.
  • Patient Management: Allows users to add, edit, and delete patient information and track appointments.
  • Mapping with Leaflet: Visualizes patient locations on a map, with options to open in Google Maps for further navigation.
  • Data Visualization: Provides charts for viewing appointment trends and other patient statistics.
  • Data Import/Export: Import and export patient data in Excel format for seamless data management.

Tools Used for Development:

  • Express: Backend framework for building the application.
  • Vite-Express: Used to streamline development with fast builds.
  • MariaDB: Database for storing patient information.
  • React: JavaScript library for creating the user interface.
  • Leaflet & React-Leaflet: Used to display patient locations on a map.
  • React-ChartJS-2: For creating interactive charts to visualize data.
  • React-to-Print & xlsx: Enables printing to PDF and exporting data to Excel format.

Monitor Server Web Application
